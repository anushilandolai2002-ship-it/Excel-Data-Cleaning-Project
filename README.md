# Data Cleaning and Analysis Project

##  Project Overview
This project demonstrates my skills in **data cleaning** and **data analysis** using Microsoft Excel.  
The goal of this project is to transform raw data into meaningful insights by cleaning errors, standardizing formats, and analyzing patterns.

##  Data Cleaning
- Removed duplicate records  
- Standardized phone numbers, names, and date formats  
- Handled missing values  
- Used Excel functions like:
  - `TRIM()`, `PROPER()`, `UPPER()`
  -  Flash Fill  
- Ensured the dataset was ready for analysis

##  Data Analysis
- Performed descriptive analysis using Excel formulas and pivot tables  
- Identified key insights and trends from the cleaned dataset  
- Created charts and summaries for better understanding  
- Applied Excel AI tools for deeper insights

##  Tools Used
- Microsoft Excel 2019 / Excel Online  
- Excel AI tools  
- GitHub for project hosting

## Files Included
- **Data_Clean.xlsx** – The raw and cleaned data process  
- **Sales Data Analysis.xlsx** – Analysis and insights derived from cleaned data  

## Objective
To build a practical project that showcases data entry, cleaning, and analysis skills for data-related roles.
